# Sleep-Health-Analysis 睡眠品質分析

## 研究目的
本研究的主要目的是找出提升睡眠品質的方法，並識別與睡眠品質直接相關的關鍵因素。我們關注的主要問題包括：

- 哪些因素會直接影響睡眠品質？
- 如何通過調整這些因素來提高睡眠品質？

研究層面分為四層：

1. **基本睡眠品質分析**  ：性別、年齡、職業、壓力程度
2. **生活方式對睡眠的影響**  ：身體活動量、步數、BMI
3. **心血管健康對睡眠的影響**  ：血壓、心率、BMI
4. **整體睡眠狀況分析**  ：顯著因素對睡眠品質的影響

## 資料來源
資料集來自 Kaggle 公開資料集 **Sleep Health and Lifestyle Dataset**，共 **13 個欄位、374 筆樣本**。

### 主要變數說明

| 欄位名稱 | 說明 |
|--------|------|
| Person ID | 人員識別碼 |
| Gender | 性別（Male / Female） |
| Age | 年齡 |
| Occupation | 職業 |
| Sleep Duration | 每日睡眠時數 |
| Quality of Sleep | 睡眠品質（1–10） |
| Physical Activity Level | 每日身體活動分鐘數 |
| Stress Level | 壓力程度（1–10） |
| BMI Category | BMI 類別 |
| Blood Pressure | 血壓（收縮壓/舒張壓） |
| Heart Rate | 靜止心率 |
| Daily Steps | 每日步數 |
| Sleep Disorder | 睡眠障礙類型 |
